# .Net Core 6.0

## 1.17 Inital Release
- Remove Newtonsoft.Json and use [System.Text.Json](https://docs.microsoft.com/en-us/dotnet/standard/serialization/system-text-json-migrate-from-newtonsoft-how-to?pivots=dotnet-6-0) instead
- Use [.Net 6.0 Minimal APIs](https://docs.microsoft.com/en-us/aspnet/core/fundamentals/minimal-apis?view=aspnetcore-6.0) and framework-based json parsing
